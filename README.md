## 📁 Tasks Completed

### ✅ Task 1 — Web Scraping
- Scraped book data from books.toscrape.com
- Extracted Title, Price, Star Rating, Availability
- Saved data to CSV using Pandas
- **Libraries:** requests, BeautifulSoup, pandas

### ✅ Task 2 — Exploratory Data Analysis (EDA)
- Performed full EDA on Titanic dataset
- Handled missing values and data cleaning
- Applied Chi-square, t-test, ANOVA hypothesis tests
- Generated univariate, bivariate and correlation plots
- **Libraries:** pandas, numpy, matplotlib, seaborn, scipy

### ✅ Task 3 — Data Visualization
- Built 9-chart dashboard using Matplotlib and Seaborn
- Created interactive Plotly dashboards
- Built world choropleth map for Happiness dataset
- **Libraries:** matplotlib, seaborn, plotly
